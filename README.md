# El Mundo del Mueble · Web

Sitio de El Mundo del Mueble (V10.4) — HTML autocontenido.

## Publicar con GitHub Pages
1. Subí todos los archivos de esta carpeta a la raíz del repositorio (o a `/docs`).
2. En el repo: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / `/ (root)`** → Save.
3. En 1-2 minutos queda online en `https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`.

## Notas
- `porque-atencion.jpg`, `porque-entrega.jpg`, `porque-trayectoria.jpg` son opcionales (sección "Por qué elegirnos"): si no están, se muestran fotos stock automáticamente. Cuando tengas las imágenes IA, subilas con esos nombres.
- Los términos legales son borrador profesional: revisión de abogado antes del lanzamiento comercial.
