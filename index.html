⚙ Configurar Mara IA

Para que Mara responda con IA real, pegá tu API key de Anthropic. Se guarda solo en este navegador.

🍌 Nano Banana · Ambientación visual

Google Gemini 2.5 Flash Image (Nano Banana) toma fotos de producto sobre fondo blanco y las re-renderiza dentro de ambientes diseñados. Una imagen de catálogo de fábrica → foto profesional ambientada en segundos.

¿Cómo conseguir las API keys?
Anthropic (Claude): console.anthropic.com → API Keys → Create
Google (Nano Banana): aistudio.google.com/apikey → Create API key

Sin API keys: Todo funciona con datos simulados realistas para que veas el flujo igual.
